# Offline packages

Эта папка предназначена для данных Web/PWA версии БТКА 8.1.

На первом шаге здесь лежит только `app-shell.json`, чтобы проверить механизм offline-подготовки. Полные данные приложения будут добавлены после выбора формата хранения:

- статические справочники и тексты;
- изображения упражнений и полезностей;
- стартовая база SQLite/IndexedDB;
- манифест с контрольными суммами и размером загрузки.
