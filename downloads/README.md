# Distribution files

Сюда будут помещаться файлы для первого этапа:

- `BTCA-Windows-8.1-Setup.exe` - установщик Windows из проекта `D:\Projects\BTCA_app`;

Пока файлы не добавлены, Web-интерфейс показывает эти варианты как подготовленные места для подключения дистрибутивов.
